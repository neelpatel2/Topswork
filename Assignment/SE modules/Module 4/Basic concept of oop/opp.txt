What is Opp? List out concepts of OPP?
Ans 
OPP means Object oriented programming is a programming paradigm that revolves around creating "objects" to model real-world entities.
These objects encapsulate data (attributes) and the functions (methods) that operate on that data.
Concepts
•	Object
•	Classes
•	Inheritance
•	Encapsulation
•	Abstraction
•	Polymorphism
